Plataforma de Mentoría Académica

"Conexión Educativa" facilita la mentoría entre estudiantes de ciclos iniciales y avanzados, optimizando el desarrollo académico y profesional mediante un ecosistema colaborativo.

Explorar Features

Descripción General

Nuestra plataforma integra un sistema de matching basado en especialidades, gestión de sesiones y un mecanismo de retroalimentación para garantizar la calidad de las mentorías y el éxito académico.

  • Matching Inteligente: Conectamos aprendices con mentores ideales
  • Gestión Integral: Agenda, recordatorios y seguimiento automatizado
  • Calidad Garantizada: Sistema de valoraciones transparente

Funcionalidades Clave Implementadas

1. Autenticación Unificada

Registro e inicio de sesión mediante OAuth 2.0 con Gmail, asegurando credenciales válidas y reduciendo barreras de acceso.

2. Perfiles Dinámicos

Creación de perfiles diferenciados para mentores (con especialidad, experiencia y disponibilidad) y aprendices (con intereses y objetivos).

3. Búsqueda Intuitiva

Filtrado de mentores por área de conocimiento, horarios y calificaciones, con resultados en tarjetas interactivas.

4. Agendamiento Automatizado

Integración con Google Calendar para reservar sesiones, con recordatorios por correo electrónico.

5. Sistema de Valoraciones

Evaluación post-sesión mediante estrellas y comentarios, visible públicamente para transparencia.

MVP Escalable

Preparado para escalar con features como videollamadas o análisis de datos de mentorías.

Seguridad y Usabilidad

Protección de Datos

Cumplimiento de protección de datos personales bajo estándares OWASP, garantizando la seguridad de la información de todos los usuarios.

Accesibilidad Universal

Interfaz accesible (WCAG 2.1 Nivel AA) compatible con dispositivos móviles y escritorio, asegurando una experiencia inclusiva.

Flujo de Usuario

1

El aprendiz busca un mentor

Según sus necesidades académicas, área de conocimiento y disponibilidad horaria.

2

Visualiza perfiles y selecciona

Explora tarjetas interactivas con información detallada del mentor y agenda una sesión.

3

El mentor recibe y confirma

La solicitud se sincroniza automáticamente en ambos calendarios de Google.

4

Calificación y retroalimentación

Tras la sesión, el aprendiz califica la experiencia, retroalimentando el sistema.
